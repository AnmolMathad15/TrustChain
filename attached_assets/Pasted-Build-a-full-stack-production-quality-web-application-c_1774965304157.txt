Build a full-stack production-quality web application called:

"SSI Platform (Single Smart Interface)"
— a unified AI-powered platform for accessing essential government, financial, healthcare, and identity-based services in one place.

The platform should be designed for both:
1) Urban smartphone users
2) Rural kiosk-based users with low digital literacy

---

⚡ TECH STACK:

Frontend:
- React (Vite) + Tailwind CSS + ShadCN UI
- Framer Motion (animations)

Backend:
- Node.js + Express

Database:
- Firebase (Firestore + Authentication)

AI Layer:
- OpenAI API (chat + intent detection)
- Web Speech API (speech-to-text + text-to-speech)

---

🎯 CORE PRODUCT ARCHITECTURE:

The system should be modular and structured into layers:

1. Identity Layer
2. AI Assistant Layer
3. Service Modules Layer
4. Security Layer
5. Personalization Layer

---

🔐 1. IDENTITY LAYER:

- Firebase Authentication (OTP simulation)
- User profile:
  - Name
  - Phone
  - Preferred language
- SSI ID (unique user ID)
- Simple PIN-based re-authentication (mock)
- Family linking (basic UI simulation)

---

🤖 2. AI ASSISTANT (CENTRAL BRAIN):

- Floating AI assistant (bottom-right)
- Accept:
  - Text input
  - Voice input (mic button)

- Capabilities:
  - Intent detection (route user to modules)
  - Task-based commands:
    Examples:
    - "ATM kaise use kare?"
    - "Mera Aadhaar dikhao"
    - "Form bharna hai"
    - "Electricity bill bhar do"
    - "Meri health report explain karo"

- AI responses:
  - Simple, multilingual
  - Voice output + text output

---

🏠 3. DASHBOARD (SUPER APP UI):

- Clean, modern UI with large accessible cards:
  - 🏧 ATM Assistant
  - 📄 Documents
  - 📝 Government Forms
  - 🏥 Healthcare
  - 💸 Payments
  - 📢 Govt Services
  - 🔔 Notifications

- Sidebar navigation (collapsible)
- Top ribbon with app branding
- Dark + Light mode

---
